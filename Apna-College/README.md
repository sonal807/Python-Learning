## 2️⃣ Variables & Data Types

Variables are used to store data in memory. Python automatically detects the data type of a variable.

### Topics Learned

- Variables
- Integer
- Float
- String
- Boolean
- Type Conversion
- Type Casting

```python
name = "Sonal"
age = 21
marks = 87.5
```

📝 **Key Notes**

- Variable names cannot start with numbers.
- Python is case-sensitive.
- Data types are assigned automatically.

Operators perform different operations on variables and values.

### Topics Learned

- Arithmetic Operators
- Relational Operators
- Assignment Operators
- Logical Operators
- Membership Operators
- Identity Operators

## 5️⃣ Conditional Statements

Conditional statements allow programs to make decisions.

### Topics Learned

- if
- elif
- else
- Nested if

```python

    print("Eligible")

```

📝 **Key Notes**

- Every condition evaluates to either **True** or **False**.

## 6️⃣ Loops

Loops execute a block of code repeatedly.

### Topics Learned

- for loop
- while loop
- range()
- break
- continue
- pass

```python

    print(i)
```

📝 **Key Notes**

- `break` → exits the loop.
- `continue` → skips the current iteration.
- `pass` → placeholder statement.
